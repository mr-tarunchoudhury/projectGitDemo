# Learning How to push local repo to git hub
# step1
go to path of directory on terminal and run command init

# step2
create a new repo in git hub

# step3
copy the HTTPS link and in terminal run command git remote add origini "link"
uu can also verify the origin, just run command git remote -v

# step4
change the git branch name to main: git branch -M main

# step5
now add all the files to stage: git add . and then commit: git commit -m "message"

# step6
now push the file: git push origin main
